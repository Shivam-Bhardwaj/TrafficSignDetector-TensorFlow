Traffic Sign Detector Tutorial

This tutorial provides a step-by-step guide to setting up, training, and using the traffic sign detector.

1. Introduction

This project is a deep learning application that classifies German traffic signs. It uses a Convolutional Neural Network (CNN) built with TensorFlow.

2. Setup and Installation

Prerequisites

  • Pip
  • Python 3
  • Virtual Environment

Install instructions

open terminal

bash $ git clone https://github.com/Shivam-Bhardwaj/TrafficSignDetector-TensorFlow.git $ cd TrafficSignDetector-TensorFlow $ virtualenv --no-site-packages -p python3 venv $ source venv/bin/activate $ pip install -r requirements.txt

3. Running the Project

There are two main ways to use this project:

  1. Train the model from the command line.
  2. Use the web application for predictions.

Option 1: Train the Model

The run_traffic_classifier.py script allows you to train the model directly.

  1. Train the model with default settings: bash python run_traffic_classifier.py This will download the dataset, train the model for 10 epochs, and save it as lenet.keras.

  2. See available training options: bash python run_traffic_classifier.py --help

Option 2: Use the Web Application

The app.py script launches a Flask web server, allowing you to upload images and see predictions.

  1. Make sure the trained model exists. You must have a lenet.keras file in the root directory. If not, train the model first (see Option 1).

  2. Run the web application: bash python app.py

  3. Open your web browser and go to http://127.0.0.1:5000 to use the classifier.
